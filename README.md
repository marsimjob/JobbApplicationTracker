# JobbApplicationTracker

## WHAT IS IT?
---------------------------------------------------------------------------------------------------------------------------------------------------
	A console based application that let's you browse, edit and sort job applications. You move across the menus using up and down keys and enter.
	Follow the instructions to add the right information. 
---------------------------------------------------------------------------------------------------------------------------------------------------
## WHY IS IT?
---------------------------------------------------------------------------------------------------------------------------------------------------
	The project is a culmination of all things learned during the first few weeks of our courses. OOP, Collections, LINQ, Structure, Git & GitHub.
---------------------------------------------------------------------------------------------------------------------------------------------------
## HOW DID LINQ HELP THE PROCESS?
---------------------------------------------------------------------------------------------------------------------------------------------------
	There were a few occassion in the JobManager script that LINQ came in handy and made it easy to filter out information from my main collection
	variable that was the JobApplication list.
		** Checking for matches in the list to return a true or false boolean (.Any().Equals()).
		** Getting a list of filtered items out of the main list (.Where()).
		** Getting one first best item that matches the requirment (.FirstOrDefault()).
		** Calculating the Avarage of a list of integers (.Avarge()).
		** Rearranging order of items to a new list ( .OrderBy(Descending)()).
	It smoothened out the process and made sorting and displaying data in a flexiable matter very possible. Easy-To-Use baked in solutions to lists!
